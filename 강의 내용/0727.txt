ClassVariable

클래스 변수(static 변수) : 클래스 이름으로 
접근 가능한 변수

인스턴스(객체) 변수 : 객체가 생성(new Class())되어야만
 접근 가능하다.

지역 변수 : 특정 지역에서만 사용할 수 있다. 
(우리가 계속 사용한 변수)

중괄호 {} 범위를 알려줌
public 외부에 접근 허용
private 외부 접근 불가

프로그램을 동작시키기 위해 저장소에 저장
코드 작업하는 것들이 메모리 공간쪽에 올라옴
그 올라온 값을 CPU가 처리
static 키워드가 붙으면 프로그램이 시작이 하게 되면
static 키워드가 붙은 값을 모두 메모리공간에 저장
이 프로그램이 종료되기 전까지 공간을 계속 차지
없으면 new라는 연산으로 객체를 만들어 사용 

final = > 해당하는 값을 변경 할 수 없습니다.

인스턴스 변수로 만드는게 현명함
