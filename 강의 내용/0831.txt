파일 입출력 ( 기본 스트림)
	기존에 사용했던 프로그램 
	이클립스에서 실행 = > 실행파일 동작
	=> 실행되는 값들이 RAM(메모리) 에 올라감
	=> CPU가 연산함
	=> 프로그램 종료하면 RAM에 저장된 데이터 삭제
	=> 재실행하면 새롭게 프로그램이 올라감(기존 데이터 삭제)
	=> 램에 저장하는게 아닌 디스크에 저장
	연결통로 = stream
	들어오는거 input 나가는거 output
	File filePath = new File("C:/java/test/aaa.txt");
	//파일쪽으로 나가는 통로를 만들겠다
	FileOutputStream fos = new FileOutputStream(filePath, true);
	//true가 없으면 기존에 있던 파일을 파괴시키고 새로운 파일을 만들어서 내용을 작성
	//true 가 있으면 기존에 있던 내용은 유지하고 뒤에 이어붙임 
	fos.write('A');
	String s = "aaaaaaaaaaaaaaaaaaa";
	fos.write(s.getBytes()); //String 을 byte 형식으로 바꿔줌
