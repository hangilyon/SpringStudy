많은 경험이 있어야지만 응용이 된다.

변수를 만들 때 변수를 만들자마자 값을 대입 = 초기화
초기화 하지 않으면 알수 없는 쓰레기값이 들어감.

첫번째 문자가 소문자인 자료형은 0으로 초기화하면 된다. (ex : int, float ...)
첫번째 문자가 대문자인 자료형은 null; 으로 초기화한다. (ex : String)

if :참인 경우 종속문장 실행
else : if의 조건이 거짓일 경우 실행
if와 else는 항상 짝을 이뤄야 한다.
if의 종속문장 다음에 else가 올 수 있다.

if else 조건식

else if 조건식
if문은 단독으로 봄
참이든 거짓이든 다음 문장을 봄
그러므로 if문장 밑에 if를 적으면 겹치는 값이 나오므로 여러개 실행됨.
그러므로 else if 를 넣으면 겹치는 값이 없어서 중에서 하나만 실행됨.
