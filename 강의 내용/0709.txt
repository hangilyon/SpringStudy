자료형이란 게 필요
문자열을 저장할때는 "" 숫자는 그냥
정수를 저장을 하겠다 short int [보편적]
byte : 컴퓨터에 단위 bit 8개 = 1byte
실수 : 소수점이 있는 숫자

정 수 : short : 2byte 저장용량이 작다 : -32768 ~ 32767
정 수 : int : 4byte 저장용량이 크다 : 양수 음수 대량 21억 표현
실 수 : float : 4byte : 양수 음수
실 수 : double : 8byte : 양수 음수
문 자 : byte : 1byte => 8bit : 양수 음수 표현
문 자 : char : 2byte => 16bit : 양수의 값만 표현

bit = 2진수에 대한 1자리
win r 실행 calc 계산기 프로그래머 용으로 바꿈
HEX 16진수
DEC 10진수
OCT 8진수
BIN 2진수
1byte 저장용량 0 ~ 255             0000 0000 ~ 1111 1111
4byte 저장용량 0 ~ 4,294,967,295
반은 음수로 반은 양수로 나눠짐

자바에서 문자는 유니코드 사용
문자와 숫자 연산이 가능한 이유는 컴퓨터가 문자를 알아 볼 수 없기 때문
